# sorcerer (소서러)

## 채택 근거 (UESP 자동 보강)
__NOTOC__

Sorcerer is a class in Elder Scrolls Online. Sorcerer is the class with the most magical abilities. It contains three skill lines, which each contain different types of spells for different roles. It is typically focused on lightning magic, similarly to the Dragonknight's emphasis on fire magic and Warden's use of ice magic.
