# Vicecanon (바이스캐넌(?))

## 채택 근거
Vicecanon이라는 호칭의 경우 '부(副)·차(次)·대리(代理)'를 의미하는 Vice-와, 머들이 쓰는 '규범, 정치'의 의미로 사용하는 Canon의 합성어로 보이는 것이에요. 알트머가 행정관을 Canonreeve라고 하는 것도 참고하려고 했는데, Reeve 자체에 지방 행정관의 의미가 있는 거라 Canon을 '행정관'이라고 번역하는 건 또 맞지 않는 것 같음. 일단 Vicecanon 호칭이 붙은 NPC들을 추적해 보니 던머 영토인 Shadowfen에서 해당 지역을 감독하는 3명의 NPC(Heita-Meen(아르고니안), Servyna(던머), Hrondar(노르드))들에게 붙은 칭호기도 하고, 해당 지역에는 Archcanon 직함을 가진 NPC도 있는 거라 '부'라는 말은 확실히 들어가도 될 것 같은데, Canon을 어떻게 해석할까 싶어서 우선 기존에 있는 번역을 존중하고자 Archcanon 자체부터 찾아보니까 이것부터가 아크캐논, 아치캐논 이렇게 중구난방이라 어떻게 해석해야 할지 모르겠네?ㅠ 우선 ESO 통일안 기타사전 참고해서 '성직자'라고 하면, 부성직자는 직관적으로 뜻 해석이 잘 안 되는 것 같고, 대리성직자, 성직자 대리 정도로 해석해 둘 테니 한번 의견을 부탁드리겠습니다. 음... 대리성직자보다는 성직자 대리라고 부르는 게 좀 더 자연스러울 것 같다. 이상하게 보이면 나중에 수정해줘!
