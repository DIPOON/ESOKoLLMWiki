# A Clothier's Primer — Shattered Masque의 신발 제작 가이드

## 본문 요약

[[blessed-crucible|Blessed Crucible]]의 *챔피언 The Shattered Masque*가 *미래 도전자들*에게 보내는 *조롱 섞인 의류 제작 튜토리얼*. **게임 내 *Clothing 직업 튜토리얼* 텍스트** + *캐릭터 풍자*.

### 단계 (게임플레이 가이드)
1. **Jute 획득** — *raw jute는 *싸고 풍부, 짜기 쉬움*. *야생의 노란 꽃 찾기*. *남에게 주워오라는 건 *게으름 — *arena에서 처벌* 대상*.
2. **Cloth 정제** — *clothing station에서 10 raw jute → refined jute*. *"loom 작업이 *너에게 비천*하다면 약자"*.
3. **Style** — *[[tamriel|Tamriel]] 각 문화별 style*. *자기 race의 style이 가장 쉬움. *시장 구매 또는 *징징대는 패배한 적*에게서 *탈취*.
4. **Shoes 제작** — *style 재료 + 5 bolt refined jute → homespun shoes*. *bolt 더 쓰면 *고품질*.

### Shattered Masque 식
> *"arena 전사라면 *단순 신발에 멈추지 않음*. *[[rawhide-armor|rawhide]] 부츠 + 화려한 [[sashes|sash]]*. *clothier 일을 익히면 — *내가 *네가 사랑한 모든 사람 앞에서 *철저히 굴욕*시킨 후* *trade로 살 수 있게*."*

### 개선 / 해체 / 연구
- **Improving**: *tannin 사용*. *부주의하면 *옷과 tannin 모두 망침*. *직접 찾아라 — 다 쉽게 안 해 줘*.
- **Deconstruction**: *코너 자르는 사람용*. *옷 파괴되나 *모자라도 건짐 — *내가 *머리에서 떨궈* 줄게*.
- **Researching Traits**: *더 나은 clothier 제품 *분해+연구로 *trait 학습*. *내가 그것을 *먼지로 짓밟기* 전까지 *자신감 부양*에 좋음*.
